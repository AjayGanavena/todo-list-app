# Todo List App

A simple and responsive **Todo List Web Application** built using **HTML, CSS, and JavaScript**.  
This app allows users to add, edit, mark as completed, delete, and filter tasks.  
All tasks are stored in the browser's **localStorage**, so the data remains even after refreshing the page.
